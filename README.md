# H3mart
H3Mart Back end task

### Initialize
> run "npm init" in the terminal


### Install
```
npm install body-parser, express, express-fileupload, xlsx
```

### Run the server
>npm start

### Step 1
```
"To upload the file in the Rest Api"
Go to "http://localhost:3000/upload" and set the method to post
```
